## 📖 Description

Plateforme éducative interactive proposant **22 exercices progressifs** pour l'apprentissage complet de l'environnement matériel d'un système informatique. Cette application web couvre l'intégralité du programme de l'Unité 1 : composants, périphériques, systèmes d'exploitation et réseaux informatiques.

🔗 **Démo en ligne** : [https://tabdellah.github.io/u1/](https://tabdellah.github.io/u1/)

## 📚 Liste complète des 22 exercices

### 🔧 Composants et périphériques (Exercices 1-5)

**Exercice 1 : L'ordinateur**
- Découvrir les composants principaux d'un ordinateur

**Exercice 2 : Périphériques externes**
- Identifier les différents périphériques externes de l'ordinateur

**Exercice 3 : Fonctions des périphériques**
- Relier chaque périphérique avec sa fonction correspondante

**Exercice 4 : Entrée et sortie**
- Classer les périphériques selon leur catégorie : entrée ou sortie

**Exercice 5 : Classification complète**
- Classer les périphériques : entrée, sortie, entrée/sortie et stockage

### 🔌 Connectivité (Exercices 6-7)

**Exercice 6 : Périphériques et connecteurs**
- Associer chaque périphérique avec son connecteur approprié

**Exercice 7 : Connecteurs et ports**
- Relier chaque connecteur avec son port correspondant

### 📏 Unités de mesure (Exercices 8-9)

**Exercice 8 : Unités de mesure**
- Classer les unités de mesure de la plus petite à la plus grande

**Exercice 9 : Les unités de mesure**
- Approfondir la compréhension des unités de mesure informatiques

### 🛠️ Composants internes (Exercices 10-12)

**Exercice 10 : Composants internes**
- Identifier visuellement les composants internes de l'ordinateur

**Exercice 11 : Fonctions des composants**
- Associer chaque composant interne à sa fonction correcte

**Exercice 12 : Nommer les composants**
- Regarder l'image du composant et écrire son nom correct

### 💾 Traitement et architecture (Exercices 13-14)

**Exercice 13 : Traitement des données**
- Comprendre le parcours des données dans l'ordinateur (exemple : appui sur une touche)

**Exercice 14 : Système informatique**
- Découvrir l'architecture globale d'un système informatique

### 💻 Systèmes d'exploitation (Exercices 15-18)

**Exercice 15 : Bureau Windows**
- Se familiariser avec l'environnement du bureau Windows

**Exercice 16 : Éléments du bureau**
- Identifier les différents éléments présents sur le bureau

**Exercice 17 : Systèmes d'exploitation**
- Reconnaître les différents systèmes d'exploitation

**Exercice 18 : Système et appareil**
- Associer chaque système d'exploitation à son appareil

### 🌐 Réseaux informatiques (Exercices 19-22)

**Exercice 19 : Équipements réseau**
- Découvrir les différents équipements utilisés dans un réseau

**Exercice 20 : Équipement LAN**
- Identifier les équipements spécifiques d'un réseau local (LAN)

**Exercice 21 : Nom des équipements LAN**
- Nommer correctement les équipements d'un réseau local

**Exercice 22 : Mots croisés du réseau**
- Résoudre des mots croisés sur le vocabulaire des réseaux locaux

## 📦 Installation locale

```bash
# Cloner le repository
git clone https://github.com/tabdellah/u1.git

# Accéder au dossier du projet
cd u1

# Ouvrir index.html dans votre navigateur
```

Aucune installation de dépendances n'est nécessaire. Le projet fonctionne entièrement côté client.

## 🎯 Guide d'utilisation

1. **Accédez** à la page d'accueil
2. **Parcourez** les 22 exercices organisés par thématique
3. **Sélectionnez** un exercice
4. **Cliquez** sur "Démarrer" pour ouvrir l'exercice dans une nouvelle fenêtre
5. **Suivez** les instructions spécifiques à chaque exercice
6. **Validez** vos réponses et progressez à votre rythme

## 🎓 Contexte pédagogique

Ce projet couvre l'intégralité de la **Première Unité : Environnement matériel d'un système informatique** du programme officiel d'informatique au collège au Maroc. Les 22 exercices sont conçus pour respecter la progression pédagogique recommandée et les objectifs d'apprentissage du ministère.

### Compétences développées
- Identification des composants matériels
- Classification des périphériques
- Compréhension des systèmes d'exploitation
- Maîtrise des unités de mesure informatiques
- Connaissance des réseaux locaux (LAN)
- Vocabulaire technique en informatique
